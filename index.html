About Me

Much of my interest in web development stems from a vehement passion for creating an unforgettable user experience. I have a love for whitespace, sleek effective code, and a well-told story; be it fact or fiction.

I run a blog and YouTube channel as well. You can read, or watch, more about me.

Some of my projects

Using decoupled architecture, the Adjective RPG engine focuses on integrating a flawless development experience with tools and features built in to both the server and the front-end website. The server is built as a light-weight Sinatra API, while the front-end uses Angular and Node.js to deliver an unparalelled user experience. Check out the Github repo for more information.

Smooth Transaction was the final project my group devised and constructed at Dev Bootcamp. It is an app that is focused on providing non-profit organizations with a simple way to receive donations coupled with D3 data visualization.

This is a simple role-playing game that I built using Ruby and Javascript during the sixth week of the on-site intensive at Dev Bootcamp. The project is based on killing the boss in each area. As of right now, only the Ranger class is done. Try it out! Be sure to read the guide once you log in, though. It will save you a ton of time, since there is a little gimmick to get started.
